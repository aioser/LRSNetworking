## Installation

JMEncryption is available through [CocoaPods](https://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'JMEncryption'
```

```ruby
pod 'JMEncryption/base64'
```

```ruby
pod 'JMEncryption/AES'
```

```ruby
pod 'JMEncryption/RSA'
```
